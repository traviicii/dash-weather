## Round 2 Active Set

The active refinement round lives in `assets/logo-options/round-2/`.

### Concept Map

- `true-north-*` — compass needle / heading instrument
- `heading-strip-*` — heading tape / readout strip
- `instrument-index-*` — gauge ticks / calibration index
- `front-trace-*` — forecast trace / front movement

### File Pattern

Each concept includes:

- `*-lockup.svg`
- `*-symbol.svg`
- `*-lockup-mono.svg`
- `*-symbol-mono.svg`
- `*-lockup-reversed.svg`
- `*-symbol-reversed.svg`
